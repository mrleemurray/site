# Swisscom

swisscom

An all-in-one tool to develop the site menu across multiple screen sizes for swisscom.ch

Built with web technologies, we created a tool to prototype & showcase CSS animations for the new swisscom.ch responsive menu system. By dragging the handle on the right of the container to change the width, the client could quickly test various animations across a range of screen sizes.
