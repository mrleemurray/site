# Status Board

Status Board

An ambient display SYSTEM to surface GLANCEABLE information in studio environments

A Vue.js web app that displays configurable information in a non-intrusive manner. Suited for viewing at 6 - 10ft away, this system can be configured through the use of YAML files. The ambient background animation is powered by WebGL.

View fullsize

Original UI.

**Technology:** Vue.js / Azure Functions / WebGL / Raspberry Pi
